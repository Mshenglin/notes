#### 题目描述

```
从上到下按层打印二叉树，同一层的节点按从左到右的顺序打印，每一层打印到一行。
例如:
给定二叉树: [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7
返回其层次遍历结果：

[
  [3],
  [9,20],
  [15,7]
]。
```

#### 解题思路

这与第一题的思路基本相同，都是二叉树的层序遍历，要注意的是如何准确的将每层的元素存入对应的列表中。是通过记录队列中的元素来实现。具体看代码注释。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        //创建list用来保存返回的元素
        List list=new ArrayList<>();
       //判断特殊情况
        if(root==null){
            return list;
        }
        //队列用来BFS用
        Queue<TreeNode> queue=new LinkedList<>();
        //由于未知二叉树节点的数量，所以先将元素存入该ArrayList中
        //二叉树的层序遍历
        queue.add(root);
        while(!(queue.isEmpty())){
            int size=queue.size();
            //临时的list用来存放每层的元素
             List<Integer> temp=new ArrayList<>();
             //通过队列中已有的元素来记录该层有几个元素
            for(int i=0;i<size;i++){
            TreeNode cur=queue.poll();
            temp.add(cur.val);
            if(cur.left!=null){
                queue.add(cur.left);
            }
            if(cur.right!=null){
                queue.add(cur.right);
            }
            }
            //将临时元素放入总的list中
            list.add(temp);
        }
        return list;
        }
}
```
